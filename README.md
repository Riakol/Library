# Web-Приложение Library

Приложение для хранения информации о книгах.

 * ## Stack:
   + Python3;
   + Django 4;
   + В качестве БД - .XLSX;
   + Jinja
   + Bootstrap 5
   + Excel (openpyxl). Разбивка (пагинация) по 5 постов на страницу.
